About this Vault:

This Vault is meant to archive and display official art mady by Tom Fischbach mostly related to his comic Twokinds. It also provides some quality of life features such as the ability to search by title and by tags and order results by date

This work was inspired by the Twokinds Gallery by Illiar which also hosts Tom official patreon pieces and also some fan art

Sources and code

In case you are interested this gallery is free and open source and you could probably use it for most art. Feel free to check the github page for it: ArtVault Gallery. Feel free to contribute or raise issues. The scrapper that powers the gallery is also open source, so you are welcome to check it out, use it fork it or whatever: Scrapper. I built it mostly to download Tom patreon sketches but you should be able to adapt it to any other patreon artist you follow withouth much issue

Usage, tips and help

This was built with simplicity in mind both in its construction and in its usage. You got two text boxes one allows you to search by Title name and the other by tags. You can also select to order the results by date either descending or ascending. Just hit query once you are ready and the system should fetch the results for you. Page is designed to load all the results as you scroll down but there is also a button for force rendering more pieces in case it bugs out.

For example if you want to search for art pieces which contain the words Good Morning in the title you simply type this on the first textbox

Good Morning

If on top of that you want to refine to find all the instances of Good Morning sketches with Flora just add the tag for the second textbox

Flora

Tags can be added to the second box by separating them with a comma, for example, to search for all pieces tagged with Keith and Natani input the following in the tags textbox:

Keith, Natani

Both searches are case insensitive so do not worry about that. For a comprehensive list of all tags consult patreon: Collections tab on Tom page

Results show a small preview of the image, if you click it you can get a shareable link at full resolution and the PAtreon logo will take you to the source
